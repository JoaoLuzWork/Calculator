# Calculator
 
 Calculator — Live Web Application

 A fully functional web-based calculator built with vanilla JavaScript, HTML, and CSS, deployed live on Vercel. The project demonstrates clean front-end architecture with a clear separation of concerns (separate /scripts and /styles folders).

Key features:
- Live, deployed application accessible via Vercel
- Standard arithmetic operations with a responsive interface
- Vanilla JavaScript implementation — no frameworks or libraries
- Organised project structure with separated logic, styling, and markup
